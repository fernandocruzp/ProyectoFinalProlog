# Busqueda del tesoro
En este juego tienes que encontrar el tesoro de la verdad, oculto por Mildivar el rey que gobierna con paz, buscando en 20 ubicaciones distintas, la ayudante del rey Amoni te dará pistas si estás muy lejos.

## Instrucciones de uso
Usa consultas para elegir un lugar de los 20 disponibles, el juego responderá si es correcto o no
