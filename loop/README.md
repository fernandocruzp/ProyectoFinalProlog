Nombre del juego: Loop.

Descripcion:
Es un juego entre dos jugadores. El tablero es una grafica completa K6, en el cual los jugadores se alternan por colocar aristas (conexiones) de su propio color. El objetivo es evitar formar un ciclo de 3 vertices con aristas de un mismo color.
